# 使用文档

##初始化

``` bash
# install dependencies  安装依赖
npm install

# serve with hot reload at localhost:8080  运行程序
npm run dev

# build for production with minification 打包构建
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
